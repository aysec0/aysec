Tools.

Open-source tools I've built or maintain. Free, MIT-licensed, written because I needed them on a real engagement and decided to clean up the result for the rest of the community.

jwt-fuzz

python

A JWT fuzzer that catches all the alg/key confusion bugs in one pass. Detects alg=none, weak HMAC keys, RSA→HMAC confusion, JKU/JWK injection. Built because every existing tool only catches a subset.

★ 1.2k v2.1.0

recon-pipe

go

A continuous-recon orchestrator. Subfinder → httpx → nuclei → diff-vs-yesterday, runnable as a single binary. Outputs to JSON, sqlite, or markdown reports. Built for bug-bounty automation.

★ 840 v1.3.4

aws-trail

python

Walks an AWS account looking for the 5 IAM/S3/IMDS misconfigs that cause 80% of real-world breaches. Read-only. Output is a heat-map you can hand to engineering on day 1.

★ 670 v0.9.0

prompt-eval

python

A red-team eval harness for LLM apps. Pluggable adversarial prompts, scoring across direct/indirect injection + jailbreak axes, CI-friendly JSON output. Used in AI Red Teaming.

★ 2.4k v3.0.0

ad-acl-walk

python

BloodHound-friendly ACL collector that handles the edge cases in modern AD environments (cross-forest trusts, ADCS templates, inherited rights via security groups).

★ 510 v1.1.2

mobile-pinned

bash

Universal cert-pinning bypass scripts for Frida + Objection, covering Android (OkHttp/TrustKit/native pinning) and iOS (NSURLSession/AFNetworking/TrustKit). One command.
